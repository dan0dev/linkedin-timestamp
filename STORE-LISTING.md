## Detailed Description

**Stop guessing when LinkedIn posts were actually published!**

LinkedIn shows vague relative times like "2h ago", "1d ago", or "3w ago" - making it impossible to know exactly when something was posted. This extension reveals the **precise timestamp** hidden in every LinkedIn post and comment.

### üîß How It Works

LinkedIn uses "Snowflake IDs" - special numerical identifiers that encode the timestamp when the post was created. This extension:

1. Extracts the post/comment ID from LinkedIn pages
2. Decodes the embedded timestamp using mathematical operations
3. Formats the date according to your locale settings
4. Displays it right where the relative time used to be
