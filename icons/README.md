# Icon Placeholders

Create three PNG icon files with the LinkedIn logo or a clock/calendar symbol:

- **icon16.png** - 16x16 pixels
- **icon48.png** - 48x48 pixels
- **icon128.png** - 128x128 pixels

Suggested design: Blue background (#0a66c2) with a white clock or calendar icon.

You can use any icon generator or create them manually. Place all three files in this `icons/` directory.

## Quick Icon Generation

You can use online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- Or any image editor to create simple icons

For a quick solution, you can use a solid color or simple text-based icon for testing.
